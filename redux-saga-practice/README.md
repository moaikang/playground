# ✏️ Redux-saga practice

Redux-saga를 학습하고, 삽질합니다.  
간단한 TodoList를 만들면서 세 가지를 알아가려 합니다.  
Redux-saga를 학습하는 것에 초점이 맞춰저 있으므로 TS는 우선 사용하지 않습니다.

## Install

**Front-End**

```bash
  cd front
  npm i
```

**Back-End**

```bash
cd back
npm i
```

## Start

**Front-End**

```bash
  cd front
  npm start
```

실행한 뒤, localhost:3000으로 접근할 수 있습니다.

**Back-End**

```bash
cd back
npm i
```

8000 포트로 실행됩니다.
